# 用户管理

# 系统管理员

  1. 点击用户管理——可以查看当前系统中所有用户
  2. 系统管理员拥有最高的系统权限

# 创建平台权限

  1. 点击用户管理——创建平台权限——可以设置用户创建平台权限
  2. 用户只能在权限内创建平台

# 使用平台权限

  1. 点击用户管理——使用平台权限——可以设置用户使用平台权限
  2. 用户只能在权限内使用勾选的平台

# 应用模块权限

  1. 点击用户管理——应用模块权限——可以设置用户应用模块权限
  2. 用户只能在模块权限内，给权限内的平台，设置授权模块
