# Installation Guide

# Environmental requirements

Installation environment requirements: PHP7.4+Mysql (5.7 recommended) +Nginx/Apache

# Installation procedure

  1. Download the install.php (opens new window) file to the root directory of the website
  2. Cancel the php7.4 disable function -- exec
  3. Access https:// your domain name /install.php to install the system
  4. Configure your database and admin password as prompted
  5. Add the configuration at the end of the server file /etc/ssh/ssh_config
    Include /www/wwwroot/sunphp.git/*.conf
