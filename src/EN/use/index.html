# Introduction

sunphp (opens new window) is a support for multiple users, multiple platforms, application development framework. Its fundamental purpose is to realize the rapid development of public accounts, small programs, H5, APP, without the developer to repeat the wheel! Empower every developer!

# Technical support

vue management background, thinkphp6 multi-application mode development, installation environment requirements: PHP7.4+Mysql (recommended 5.7) +Nginx/Apache

# Function supported

Built-in commonly used wechat pay, Alipay pay, file upload, seven cattle cloud storage, Ali cloud OSS, Tencent cloud COS, wechat login, Ali cloud SMS, Tencent cloud SMS, email sending and other functions, developers only need to view /app/demo/controller directory examples, control can be used!

Compatible support The module created in the /addons/ directory serves as the compatibility function and supports the compatible running of the engine 2.0 module. (Howeveris not recommended for new application development)
