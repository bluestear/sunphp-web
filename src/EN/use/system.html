# System Settings

# Site Settings

  1. Click System Settings -- Site Settings -- to set the site information
  2. You can also modify registration Settings and record Settings on the current page

# Payment Configuration (global)

  1. Click System Settings -- Payment Settings -- to set payment information
  2. The current configuration information takes effect globally. Unless the platform is separately configured with**, the global configuration information will be overwritten

# Storage Configuration (global)

  1. Click System Settings -- Storage Configuration -- to set the storage information
  2. The current configuration information takes effect globally. Unless the platform is separately configured with**, the global configuration information will be overwritten

# SMS Configuration (global)

  1. Click System Settings -- SMS Configuration -- to set SMS information
  2. The current configuration information takes effect globally. Unless the platform is separately configured with**, the global configuration information will be overwritten

# Mail Configuration (global)

  1. Click System Settings -- SMS Settings -- to set email information
  2. The current configuration information takes effect globally. Unless the platform is separately configured with**, the global configuration information will be overwritten
