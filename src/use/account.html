# 平台管理

# 创建平台

  1. 点击平台管理——创建平台——新建平台

  2. 支持多种平台，平台可以禁用、删除、置顶

  3. 设置平台类型，appid等信息

# 授权模块

  1. 点击平台管理——应用模块管理——为平台授权模块
  2. 该平台只能使用被授权的模块

# 授权使用者

  1. 点击平台管理——使用者管理——为平台授权使用者
  2. 该平台只能使用被授权的用户使用

  1. 点击添加使用者

  1. 设置使用者角色——操作员仅能操作平台，平台管理员可以授权模块，授权使用者

# 使用者菜单权限

  1. 点击权限——为使用者设置模块的菜单权限
  2. 默认——操作员拥有所有的菜单权限，如需修改权限，可以自主设置
  3. 勾选菜单权限——确认保存既可

# 进入平台

  1. 点击平台管理——新建平台的红框内——进入平台

# 平台配置

  1. 点击进入平台——自主设置平台的配置信息
  2. 如果不设置平台配置信息——那么将默认启用系统的配置信息
  3. 该配置仅在当前平台下生效
